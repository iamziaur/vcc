
# Chapainawabganj Voting Centers Web App

An ultra-lightweight, high-performance static web application designed for Chapainawabganj citizens to find voting center locations and emergency contacts.

## Features
- **Seat-wise Filtering:** Browse centers for Chapainawabganj 1, 2, and 3.
- **Smart Redirects:** Directly opens Google My Maps for specific unions/thanas.
- **Emergency Hub:** Quick call features for SP office and all Thanas.
- **Bangla/English Support:** Fully localized for ease of use.
- **Dark/Light Mode:** Comfortable browsing in any lighting condition.
- **Login Gate:** Secure internal access with password `mrm2022`.

## One-Click Deployment
1. Push this code to a GitHub repository.
2. Connect the repository to [Vercel](https://vercel.com).
3. Vercel will automatically detect the static setup and deploy.

## Technical Specs
- **Architecture:** React 18 + TypeScript + Tailwind CSS (via CDN).
- **Bundle Optimization:** Zero heavy images; all icons are SVG/FontAwesome.
- **Performance:** Designed to load in <300ms on 4G networks.
